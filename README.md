# The tech art wood

This repo will contains different experiment and algorithms immplementation related to graphics programming, technical art and physics simulation
I have no ambition for these projects to be re-used by other people. It is just a place for me to study and learn. But still, feel free to take what you want if it helps you to learn too.

## Projects in progress (WIP)

- SphereTracing and SDF
  - The objective is to implement **SDF Nodes** that can be easily be added to a **raymarched volume node** and that interact with each other. 
- Volumetric clouds
  - Simple volumetric clouds
  - volumetric clouds inside godot's volumetric fog
  - Volumetric clouds rendered using sdf for shapes and performance improvments
